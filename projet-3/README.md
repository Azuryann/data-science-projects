# Projet 3 – Prédiction de la consommation énergétique des bâtiments

## Contexte

La ville de Seattle souhaite réduire son empreinte carbone et anticiper la consommation énergétique de ses bâtiments.

## Objectif

Construire un modèle prédictif de la consommation énergétique.

## Données

- Données de consommation énergétique de bâtiments
- Variables techniques et environnementales

## Méthodologie

- Nettoyage et préparation des données
- Feature engineering
- Sélection de variables
- Modélisation (régression)
- Évaluation des performances

## Outils utilisés

- Python
- Pandas / NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Résultats

- Modèle prédictif performant
- Identification des variables influentes
- Interprétation des résultats

## Compétences développées

- Machine Learning supervisé
- Feature engineering
- Évaluation de modèles
- Analyse d’impact environnemental
