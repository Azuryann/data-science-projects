# Projet 2 – Préparation de données pour un organisme de santé publique

## Contexte

Santé publique France souhaite exploiter des données alimentaires pour développer une application.

## Objectif

Préparer et analyser un jeu de données nutritionnelles afin de le rendre exploitable.

## Données

- Données Open Food Facts
- Données nutritionnelles produits

## Méthodologie

- Nettoyage des données (valeurs manquantes, incohérences)
- Sélection des variables pertinentes
- Analyse exploratoire (EDA)
- Détection d’outliers
- Analyse multivariée (ACP)

## Outils utilisés

- Python
- Pandas / NumPy
- Matplotlib / Seaborn
- Scikit-learn

## Résultats

- Dataset nettoyé et structuré
- Identification de variables pertinentes
- Visualisation des relations entre variables

## Compétences développées

- Data cleaning avancé
- Analyse exploratoire
- Réduction de dimension
- Interprétation statistique
